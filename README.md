# 老蔡深夜食堂 

一个帮你戒掉深夜点外卖冲动的假装点餐应用。

灵感来自韩国爆火的"多巴胺网站"趋势。

## 功能

- 🍜 20+ 中国餐厅，100+ 菜品
- 🛒 完整的点餐流程（浏览→加购→结算→配送追踪）
- 📍 虚拟配送动画（骑手在地图上移动）
- 💰 统计省了多少钱和卡路里
- 🏆 成就系统
- 🎨 多主题色切换
- 📱 移动端适配

## 使用

直接在浏览器打开 `index.html` 即可使用。

所有数据保存在浏览器本地存储中。

## 部署

### Render 部署

1. 推送代码到 GitHub
2. 在 Render 创建新的 Static Site
3. 连接 GitHub 仓库
4. 部署

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
